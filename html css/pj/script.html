  • Events:- Javascript में Events ये important हिस्सा है | Javascript Events ये HTML के साथ परस्पर सम्बन्ध रखता है |

    Events ये DOM(Document Object Model) का एक हिस्सा है | HTML Elements पर attribute के रूप में इनका use किया जाता है |

    जब Javascript Events में Element's Attribute के through कोई भी javascript का code execute किया जाता है |

    जैसे कि,

    जब user Element पर click करता है | जब User mouse से double click करता है | जब Element mouse hover किया जाता है | जब keyboard की key press की जाती है |

    जब User कोई भी Event का प्रकार HTML Element में attribute के रूप में use करता है और जब HTML Document पर वो Event active हो जाता है तब उसपर value के रूप में दिया हुआ code; execute हो जाता है |

    For Example,

    यहाँ पर जब 'Click Me!' पर click किया जाएगा तो 'myDiv' id के Element में 'Hello World' ये message display होगा |

    Document में onclick ये event जब active हो जाती है तब 'myDiv' id के element में 'Hello World' ये message display होता है |

    < div onclick="document.getElementById('myDiv').innerHTML='Hello World'">Click Me!< /div> < div id="myDiv">< /div>



    Use More than One Events in Element :-

    Element पर एक से ज्यादा भी Events लिए जा सकते है |

    Example में जब button पर Click करके hold किया जाएगा तब 'onmousedown' ये event active होगा और mouse click को release या छोड़ दिया जाएगा तब 'onclick' event active होगा |

    < button onclick="document.getElementById('myDiv').innerHTML='Hello World'" onmousedown="document.getElementById('myDiv').innerHTML='Hi World'">Click Me!< /button> < div id="myDiv">< /div>
