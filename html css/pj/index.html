  • Animation-name :- animation-name Property का इस्तेमाल animation name देने के लिए किया जाता है |

    Rules for animation-name Property's Value

    Animation name ये alphabetic हो सकता है | for eg. pratigya, pjajin
    Animation name ये alphanumeric हो सकता है | for eg. pratigya123 ,pj4
    Animation name के शुरुआत पर _(underscore) हो सकता है | for eg. _pjjain, _4nanu
    Animation name के शुरुआत पर numeric characters नहीं हो सकते है | for eg. 4pj, 123pratigya
    Animation name में special characters नहीं हो सकते है | for eg. pj23@, nanu#
    Animation के name के बीच में या आखिर पर -(hyphen) हो सकता है | for eg. pratigya-4, pj-

    syntax

    animation-name: keyframe name/none/initial/inherit;

    property value

    Value description
    keyframe name उस keyframe का name specifies करता है जिसे आप selector से bind करना चाहते है ।
    none Default value specifies करता है की कोई animation नहीं होगा (casecade से आने वाला animation को overwrite करने के लिए इस्तेमाल किया जा सकता है। )
    initial Default value; set की जाती है |
    inherit parent element की value को set किया जाता है |
