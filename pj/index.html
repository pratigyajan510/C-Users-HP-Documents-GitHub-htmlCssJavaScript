  • align-content :- align-content property का इस्तेमाल Browser किस तरह Flex Container के child element के बीच मे या element के arms cross -axis पर space देता है ये flex -wrap की मदत से किया जाता है |

    syntax

    align-content: strech/flex-start/flex-end/center/space-between/space-around/space-evenly/initial/inherit;

    property value

    Value description
    strech Default start-edge से end-edge तक flex item फैलाये जाते है | अगर cross-axis पर बचा हुआ space होता है तो items के बीच मे equal space होता है
    flex-start flex container के start-edge पर cross-axis से alignment होती है
    flex-end flex container के end-edge पर cross-axis से alignment होती है
    center flex container के center पर cross-axis से alignment होती है
    space-between flex container के first flex item से last flex item तक के बीच के items मे एक जैसी space होती है
    space-around flex container के first flex item से last flex item तक के बीच के items मे एक जैसे space होती है। start edge ओर first flex item मे ओर end edge ओर last flex item के बीच मे भी space दी जाती है
    space-evenly flex container के start-edge से लेकर end edge तक flex item के बीच मे एक जैसी space दी जाती है
    initial Default value . set की जाती है
    inherit parent element की value को set किया जाता है

  • Css Animation :- css Animation के लिए Animation property का इस्तेमाल किया जाता है । Animation मे element के css style का कुछ समय देकर एक से ज्यादा css style मे change किया जा सकता है ।
    Animation के लिए @keyframes rule की जरूरत होती है । Animation किस प्रकार से होनी चाहिए वो @keyframes के अंदर define किया जाता है।
    Animation ये 0% पर start होता है ओर 100% पर end होता है । starting मे background-color ये black होगा ओर 5 second के बाद background-color ये white हो जाएगा । ये प्रक्रिया infinite time होती है।

    @keyframes rule :- @keyframes पर Animation का behavior दिया जाता है। animation को @keyframes के साथ दिए हुए animation name पर store किया जाता है।
    इस animation को execute करने के लिए animation property का इस्तेमाल किया जाता है

    Value description
    name Default 'none' Animation का नाम दिया जाता है |

    dealy Optional. Default '0' animation कितने वक्त के बाद start होना चाहिए वो milliseconds या seconds में दिया जाता है |
    iteration-count Optional. Default '1' Animation कितनी बार repeat होना चाहिए वो दिया जाता है |
    direction Optional. Default 'normal' animation की direction दी जाती है|
    fill-mode Optional. Default 'none' Animation complete होने पे कहा पर स्थित होना चाहिए वो दिया जाता है|
    play-state Optional. Default 'running' Animation को run या pause करने के लिए इस्तेमाल किया जाता है |

  • Animation-name :- animation-name Property का इस्तेमाल animation name देने के लिए किया जाता है |

    Rules for animation-name Property's Value

    Animation name ये alphabetic हो सकता है | for eg. pratigya, pjajin
    Animation name ये alphanumeric हो सकता है | for eg. pratigya123 ,pj4
    Animation name के शुरुआत पर _(underscore) हो सकता है | for eg. _pjjain, _4nanu
    Animation name के शुरुआत पर numeric characters नहीं हो सकते है | for eg. 4pj, 123pratigya
    Animation name में special characters नहीं हो सकते है | for eg. pj23@, nanu#
    Animation के name के बीच में या आखिर पर -(hyphen) हो सकता है | for eg. pratigya-4, pj-

    syntax

    animation-name: keyframe name/none/initial/inherit;

    property value

    Value description
    keyframe name उस keyframe का name specifies करता है जिसे आप selector से bind करना चाहते है ।
    none Default value specifies करता है की कोई animation नहीं होगा (casecade से आने वाला animation को overwrite करने के लिए इस्तेमाल किया जा सकता है। )
    initial Default value; set की जाती है |
    inherit parent element की value को set किया जाता है |

  • Animation-timing-function :- animation-timing-function Property का इस्तेमाल bezier curve के हिसाब से animation के speed को देने के लिए किया जाता है |

    syntax

    animation-timing-function :ease/linear/ease-in/ease-out/ease-in-out/step-start/step-end/steps(int,start/end)/cubic-bezier(n,n,n,n)/initial/inherit;

    property value

    Value description
    ease Default. slow start होता है बाद में fast होता है और last में फिर slow होता है |
    linear startingसे last तक same speed से होता है |
    ease-in slow start होता है |
    ease-out fast start होता है |
    ease-in-out slow start और slow end होता है
    step-start Duration के अनुसार start से directly end पर जाता है | ये steps(1, start) के equal होता है |
    step-end Duration के अनुसार end से directly start पर जाता है | ये steps(1, end) के equal होता है |
    steps(int,start/end) first parameter पर positive number of intervals दिए जाते है और second parameter optional होता है | वहा पर start या end value दी जाती है, यह intervals के बीच की value होती है | अगर नहीं दी जाती है तो by default 'end' ये value होती है |
    cubic-bezier(n,n,n,n) चारों points के लिए 0 से 1 के बीच की value दी जाती है | ये values cubic-bezier curve को define करके animation को represent करता है |
    initial Default value; set की जाती है |
    inherit parent element की value को set किया जाता है |
