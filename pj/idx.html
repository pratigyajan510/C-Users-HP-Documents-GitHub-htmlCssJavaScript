HTML Tags

  • < !-- --> :- comment tag का use source code में comment डालने के लिए किया जाता है। comment browser में display नहीं होती हैं।
  • < !DOCTYPE> :- All HTML document declaration के साथ start होने चाहिए।
    declaration एक HTML tag नहीं है। यह browser के लिए एक "information" है कि किस document type को expect की जाए।
  • < a> :- < a> tag एक hyperlink को define करता है, जिसका use one page से another page पर link करने के लिए किया जाता है।
    < a> element की most important attribute href विशेषता है, जो link's के destination को indicates करती है।

  • < abbr> :- < abbr> tag एक abbreviation या acronym को define करता है,like "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".
  • < acronym> :- HTML 4 मे एक acronym define करने के लिए use किया जाता था लेकन अब HTML5 मे नही है। इसके बदले का use करते है ।
  • < address> :- contact information specifie करता है। address की information बताता है।
    contact information एक email address, URL, physical address, phone number, social media handle, etc. हो सकती है.
    < address> element में text आमतौर पर italic में Presented होता है, और browser हमेशा < address> element के पहले और बाद में एक line break add कर देगा।
